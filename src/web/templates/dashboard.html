Three-Layer Architecture

Layer 1 — Base Credit Spread Engine
Realistic theta decay (1.2%/day), vega impact, gamma acceleration on adverse moves. 1.0x stop loss, 21-day max hold, take profit at 30% credit. Structural bull put bias (above SMA50+200).
Layer 2 — VRP Timing Model
Only trades when Variance Risk Premium > 0 (IV > realized vol). VIX sweet spot: 13-28. Filters out ~25% of bad entries, improving win quality.
Layer 3 — Cross-Asset Regime + Events
Yield curve (2s10s), VIX momentum, SPY trend, RSI extremes. Regimes: RISK_ON / NEUTRAL / CAUTIOUS / RISK_OFF. Position sizing adjusts 0.5x-1.5x. Event filter reduces size around FOMC + quad witching.
